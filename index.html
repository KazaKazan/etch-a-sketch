Single Color: Change brush color to a custom picked color. Shortcut "1"

Random Color: The brush will paint each cell a random color. Shortcut "2"

Eraser: The brush will paint cells white. Shortcut "3"

Reset: Clears the canvas.

Save: Download your painting.

Grid: Show/Hide the grid. Shortcut "G"

Click Mode: Enables/Disables "Click Mode". In click mode, you can stop and resume painting with a mouse click. The canvas will have a dotted border while you are painting. Shortcut "C"

Size: Resizes the canvas to fit AxA cells.

Color Picker: Shows/Hides the color picker panel. Shortcut "P"

Color Dropper: Switches the current color to the selected cell's color. Shortcut "D"
